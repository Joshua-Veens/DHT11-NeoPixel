Detailed Description

DHT11 Sensor by Joshua Veens.

DHT11 temperature and humidity sensor. Reads value's of temperature or humidity every 1.3 seconds. Gives back temperature in Celcius and humidity in percentage.

Has error codes: 40 startup low signal timed out. 80 startup high signal timed out. 10 checksum was not correct. 0 Succes!

◆ read_data()

void DHT11::read_data ( )

Reads the data send by the DHT11 temperature and humidity sensor.

Sets previous data to 0 so data array is empty. Loops through all 40 bits that are sent by DHT11 sensor. Check for low signal of ~50 microseconds. Check for 0 or 1 with keeping time in microseconds. 0 = ~30 microseconds, 1 = ~70 microseconds. Set 0 or 1 in data.

◆ start_up()

void DHT11::start_up ( )

This function starts up the sensor and get it ready to send data.

Sets high 'reset' signal for 1200 milliseconds on data line. Sets low signal for at least 18 ms, 20 ms to be safe on data line. Check for low then high signal of ~80 microseconds.
